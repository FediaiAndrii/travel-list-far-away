# Travel Packing List

A simple React application for managing a packing list for trips. Built as part of Jonas Schmedtmann's Ultimate React Course.

🔗 **Live Demo:** [Travel Packing List](https://fediaiandrii.github.io/travel-list-far-away)

## Features

- Add items to the list with a specific quantity.
- Mark items as packed or unpacked.
- Delete individual items or clear the entire list.
- Sort items by input order, description, or packed status.
- Track the total number of items and the percentage of packed items.

## Tech Stack

- React (Create React App)
- CSS

## Running Locally

To run this project on your local machine:

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
